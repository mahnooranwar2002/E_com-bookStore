PROJECT NAME: Bostorek 

This is a E-Commerce using asp Dot Net core MVC .

The technology that I used in this project is given below
Frontend: HTML, CSS, JavaScript, Bootstrap
Backend: Dot Net Core MVC, C#
Database: SQL Server, Entity Framework Core

These features allow customers to interact with the bookstore, manage their account, and complete purchases.

* **Contact Form:** Users can submit queries or feedback to the store administration.
* **Profile Management:** Users can update their personal information, shipping address, and password.

## üîë Admin Features 

These features are restricted to administrators for managing the store, inventory, and users.

### 1. User and Access Management

* **View User Details:** See a list and detailed profiles of all registered users.
* **Activate/Deactivate Users:** Temporarily enable or disable user accounts.
* **Delete Users:** Permanently remove user accounts.

### 2. Catalog Management (CRUD)

* **Book Genres CRUD:** Create, Read, Update, and Delete book categories.
* **Author Management CRUD:** Create, Read, Update, and Delete author details.
* **Book Management CRUD:** Manage all aspects of book entries, including Title, Price, Description, and **Quantity**.

### 3. Order and Inventory Control

* **View Orders:** Access a comprehensive list of all placed orders, including customer details and items purchased.
* **Live Book Quantity:** Inventory levels are accurately tracked and decremented instantly upon order placement.
* **Update Order Status:** Change the status of an order (e.g., Pending, Shipped, Delivered).
